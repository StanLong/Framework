# IDEA快捷键

| 快捷键                 | 说明                                          |
| ---------------------- | --------------------------------------------- |
| ctrl+alt+l             | 格式化                                        |
| Alt+Insert             | 生成类的相关方法，如getter/setter，构造方法等 |
| psvm                   | public static void main                       |
| ctrl + shift + alt + U | 以图的方式显示项目中依赖之间的关系            |
| sout                   | System.out.println                            |
| ctrl+r                 | 查找替换                                      |
| ctrl+o                 | 重写方法                                      |
| Ctrl + Alt + B         | 查看类的具体实现代码                          |
