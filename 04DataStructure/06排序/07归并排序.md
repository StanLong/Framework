# 归并排序

![](.././doc/13.gif)

归并排序是建立在归并操作上的一种有效的排序算法。该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。若将两个有序表合并成一个有序表，称为2-路归并。

- 把长度为n的输入序列分成两个长度为n/2的子序列；
- 对这两个子序列分别采用归并排序；
- 将两个排序好的子序列合并成一个最终的排序序列。

## Java版

```java
package com.stanlong;

import java.util.Arrays;

/**
 * 归并排序
 * 先分解，再合并
 */
public class DataStructure {

    public static void main(String[] args) throws Exception {
        int[] array = {3,44,38,5,47,15,36,26,27,2,46,4,19,50,48};
        mergeSort(array, 0, args.length-1);
        System.out.println(Arrays.toString(array));

    }

    /**
     * 归并排序
     * @param array 待排序数组
     * @param left 排序段左边索引
     * @param right 排序段右边索引
     */
    public static void mergeSort(int[] array, int left, int right){

        // 分解
        if(left >= right){ // left == right 时， 即表示array分解到每个归并段里只有一个元素
            return;
        }
        int mid = (left + right)/2;
        // 迭代分解左边
        mergeSort(array, left, mid);
        // 迭代分解右边
        mergeSort(array, mid + 1, right);


        // 合并
        int s1 = left; // 第一个归并段的开始
        int s2 = mid + 1; // 第二个归并段的开始

        int[] temp = new int[right-left + 1]; //定义一个临时数组用来保存两个归并段合并后的排序结果
        int i = 0; // 临时数组的下标

        while (s1<=mid && s2 <= right){ //当两个归并段都不为空
            if(array[s1] <= array[s2]){ // 升序排
                temp[i++] = array[s1++];
            }else{
                temp[i++] = array[s2++];
            }
        }

        // 归并段合并后，仍有归并段不为空。 则合并剩下的元素到临时数组
        while (s1 <= mid){
            temp[i++] = array[s1++];
        }

        while (s2 <= right){
            temp[i++] = array[s2++];
        }

        // 合并临时数组到 array
        for (int j=0; j< temp.length; j++){
            array[j+left] = temp[j];
        }
    }
}
```
