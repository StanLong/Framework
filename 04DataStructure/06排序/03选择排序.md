# 选择排序

![](.././doc/09.gif)

选择排序（select sorting）基本思想：第一次从arr[0]-arr[n-1]中选取最小值，与arr[0]交换，第二次从arr[1]-arr[n-1]中选取最小值，与arr[1]交换，第三次从arr[2]-arr[n-1]中选取最小值，与arr[2]交换，…，第i次从arr[i-1]~arr[n-1]中选取最小值，与arr[i-1]交换，…, 第n-1次从arr[n-2]~arr[n-1]中选取最小值，与arr[n-2]交换，总共通过n-1次，得到一个按排序码从小到大排列的有序序列。

## Java版

```scala
package com.stanlong;

import java.util.Arrays;

/**
 * 选择排序
 */
public class DataStructure {

    public static void main(String[] args) throws Exception {
        int[] arr = {3,44,38,5,47,15,36,26,27,2,46,4,19,50,48};

        for(int i=0; i<arr.length - 1; i++){
            // 假定最小值
            int minIndex = i;
            int min = arr[i];
            for(int j=i+1; j<arr.length; j++){  // 遍历出最小值
                if(min > arr[j]){ // 如果假定的min不是最小值
                    // 重置最小值
                    minIndex = j;
                    min = arr[j];
                }
            }
            if(minIndex != i){ // 如果这一轮下来，i 位置上的元素已经是最小值，那就不需要交换了
                // 交换
                arr[minIndex] = arr[i];
                arr[i] = min;
            }
            System.out.println("第" + i + "轮选择排序结果: " + Arrays.toString(arr));
        }
        System.out.println("选择排序总结果: " + Arrays.toString(arr));
    }
}
```

## Scala版

```scala
package com.stanlong.scala

import scala.util.control.Breaks.{break, breakable}

/**
 * 选择排序
 */
object ForDemo {

    def main(args: Array[String]): Unit = {
        var array = Array[Int](3,44,38,5,47,15,36,26,27,2,46,4,19,50,48)

        for(i <- 0 until array.length - 1){
            var minIndex = i
            var min = array(i)
            for (j <- i+1 until array.length){
                if(min > array(j)){
                    minIndex = j
                    min = array(j)
                }
            }
            if(i != minIndex){ // 如果这一轮下来，i 位置上的元素已经是最小值，那就不需要交换了
                array(minIndex) = array(i)
                array(i) = min
            }
            println("第" + i + "次排序结果: " + array.mkString(","))
        }
        println("最终排序结果: " + array.mkString(","))
    }
}
```

## Python版

```python
# 选择排序

if __name__ == '__main__':
    lst = [3,44,38,5,47,15,36,26,27,2,46,4,19,50,48]

    for i in range(lst.__len__() - 1):
        minIndex = i
        min = lst[i]
        for j in range(i+1, lst.__len__()): # 遍历出最小值
            if min > lst[j]:
                minIndex = j
                min = lst[j]
        if i != minIndex:
            lst[minIndex] = lst[i]
            lst[i] = min
        print("第%s次选择排序: %s" %(i, lst))
    print("最终排序结果: %s" %lst)
```
