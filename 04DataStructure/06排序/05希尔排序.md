# 希尔排序

![](.././doc/10.gif)

希尔排序的基本思想：希尔排序是把记录按下标的一定增量分组，对每组使用插入排序算法排序；随着增量逐渐减少，每组包含的已排序数据越来越多，当增量减至1时，整个文件恰被分成一组，算法便终止。

```java
        // 数据移动法
        // 统计比较循环次数
        int count = 0;
        for(int step = arr.length / 2; step > 0; step = step / 2){
            count = count + 1;
            for(int i = step; i< arr.length; i++){
                int j = i;
                int temp = arr[j];
                if(arr[j] < arr[j-step]){
                    while (j - step >=0 && temp < arr[j-step]){
                        // 移动
                        arr[j] = arr[j-step];
                        j = j-step;
                    }
                    arr[j] = temp;
                }
            }
            System.out.println("第" + count + "轮排序结果: " + Arrays.toString(arr));
        }
        System.out.println("最终排序结果: " + Arrays.toString(arr));
    }
}
```

## Scala版

```scala
package com.stanlong.scala


/**
 * 希尔排序
 */
object ForDemo {

    def main(args: Array[String]): Unit = {
        var array = Array[Int](84, 83, 88, 87, 61, 50, 70, 60, 80, 99)
        var step:Int = Math.ceil(array.length / 2).toInt
        var temp:Int = 0
        var count:Int = 0
        while (step>0){
            count = count + 1
            for(i <- 0 until(step)){
                for(j <- i+step until array.length){
                    if(array(j) < array(j-step)){
                        temp = array(j-step)
                        array(j-step) = array(j)
                        array(j) = temp
                    }
                    println(array.mkString(","))
                }
                //println("第" + count + "次排序结果: " + array.mkString(","))
            }
            //println("第" + count + "次排序结果: " + array.mkString(","))
            step = step / 2
        }
        println("最终排序结果: " + array.mkString(","))
    }
}
```
